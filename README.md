# My-Portfolio 🚀

This repository contains a personal portfolio website, developed with a focus on modern web technologies. The project utilizes HTML, CSS, and JavaScript, with Tailwind CSS for efficient styling.

## About This Project 💡

This portfolio project aims to showcase web development skills in a clean and professional manner. Despite a developer's personal sentiment towards certain technologies, the project prioritizes functionality and aesthetic appeal.

## Features ✨

*   **Responsive Design:** Adapts seamlessly to various screen sizes, from desktops to mobile devices.
*   **Modern Styling:** Leverages Tailwind CSS for rapid and consistent UI development.
*   **Interactive Elements:** Enhanced user experience through JavaScript functionalities.
*   **Clean Codebase:** Well-structured HTML, CSS, and JavaScript for maintainability.

## Technologies Used 🛠️

*   **HTML:** For the structure and content of the web pages.
*   **CSS:** For styling and presentation.
*   **JavaScript:** For dynamic behavior and interactivity.
*   **Tailwind CSS:** A utility-first CSS framework for rapid UI development.

## Installation 🚀

To get a local copy up and running, follow these simple steps.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/moritzwendt/My-Portfolio.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd My-Portfolio
    ```

3.  **Install dependencies:**
    This project uses npm for dependency management. Open your terminal in the project's root directory and run:
    ```bash
    npm install
    ```

## Usage 💻

To run the development server and view the portfolio locally:

1.  **Start the development server:**
    ```bash
    npm run dev
    ```

    This command will start a local development server. You can typically access your project in your web browser at `http://localhost:3000` or the port specified in your configuration.
